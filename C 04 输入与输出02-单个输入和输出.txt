getchar()和putchar()函数

    int getchar()
        从stdin中获取一个字符, 并把它返回为一个整数.
        这里将字符定为int类型, 可以读取小于或等于0的特殊字符

        该函数不带任何参数, 从输入队列中返回一个字符
        如:
            ch = getchar();
            ch = getc(stdin);
            同
            scanf("%c", &ch);


    int putchar(int c)
        向stdout中输出一个字符c, 并返回相同的字符.
        这里将字符定为int类型, 可以输出小于或等于0的特殊字符

    注意:
        这两个函数只处理单个字符
        
    这两个函数定义在stdio.h中, 定义为共预处理器使用的宏

    因为这两个函数只处理字符, 并且每次只处理一个字符, 所以他们比更通用的scanf()和printf()函数更快, 更简洁


使用场景-消除stdin中的多余字符

    while( (ch=getchar()) != '\n' );   // 读取一行中所有的字符

    赋值运算符=的优先级小于关系运算符!=, 所以得使用圆括号括起来优先执行
    不括起来就会将获得的字符与换行符比较, 然后将关系运算符得到的0或1赋值给ch
    从stdin中获取一个字符后, 再与换行符进行比较, 如果不是换行, 执行循环, 是则跳出
    完成了读取一行中所有字符的操作


gets()和puts()函数

    char *gets(char *s) 
        从 stdin 读取一行到 s 所指向的缓冲区, 直到一个终止符或 EOF.

    int puts(const char *s) 
        把字符串 s 和一个尾随的换行符写入到 stdout.
